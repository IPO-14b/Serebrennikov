Сценарий использования 1 - Загрузка игры

Пользователь нажимает кнопку Run и начинается игра.

Используются глобальные переменные и функция инициализации игры.

function run() {
      if (inPlay == 0) {
          if (selec == 1) {
              selec = 9;
              inPlay = 1;
          }
      } else {
          if (missileY == 99) {
              missileY = 18;
          }
      }
    }
